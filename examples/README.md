# Examples

This directory is intentionally minimal.

It only keeps examples that show how another agent or automation can call AutoSecAudit as a subordinate information-gathering skill.

## Included examples

- `agent_invocation/README.md`
- `agent_invocation/run_plan.py`
- `agent_invocation/run_agent.py`

These examples demonstrate:

- preflight readiness checks
- plan-first execution
- resumable agent runs
- structured output handling

No legacy scanner demos or frontend examples are kept here.
